# 一键联调脚本使用说明（v1）

## 1. 脚本位置
- `/Users/lss/Desktop/AI_MCP/scripts/一键联调验收.sh`

## 2. 作用
自动执行并验证以下环节：

1. 启动官方服务端
2. 启动 MQTT 网关
3. 同步本机 IP 到官方参数与网关配置
4. 检查端口和接口可达性
5. 检查关键参数与当前局域网 IP 一致性
6. 检查设备最近连接记录
7. （默认）检查 MCP 路径 B/C 配置与日志痕迹  
   - 路径 C 同时兼容单模块（`mcp_endpoint`）和全模块（`server.mcp_endpoint`）模式

## 3. 用法

```bash
cd /Users/lss/Desktop/AI_MCP
./scripts/一键联调验收.sh
```

也可以在可视化控制台执行：
- 打开 `http://127.0.0.1:7099/`
- 在左侧「快捷操作」点击「一键联调验收」

可选参数：

```bash
./scripts/一键联调验收.sh --no-mcp
./scripts/一键联调验收.sh --logs 320
```

- `--no-mcp`：跳过 MCP 路径检查，仅做基础联调验收。
- `--logs 320`：MCP 日志关键字检查时，读取最近 320 行日志。

## 4. 输出结果
执行结束会输出：
- 通过数量
- 警告数量
- 失败数量
- 报告文件路径（保存在 `.cache/logs/联调验收报告-时间戳.txt`）

判定规则：
- 有失败项：脚本以非 0 退出码结束。
- 仅警告项：脚本以 0 退出码结束，并提示后续排查建议。
- 全通过：脚本返回成功。
